Binairstelsel

Het tientallig getallenstelsel

Het tientallig getallenstelsel bestaat uit de cijfers 0 t/m 9. Hiermee kan je verschillende getallen maken.

Het binair getallenstelsel

Het binair getallenstelsel is een tweetallig stelsel dat alleen maar bestaat uit de cijfers 0 en 1. Als je deze getallen in een bepaalde volgorde zet, betekenen ze getallen. Het is een code die computers gebruiken.

De cijfers 0 en 1 noem je een bit. Een reeks van 8 bits noem je een byte.

ASCII

ASCII staat voor American Standard Code for Information Interchange. Deze coderingsstandaard is een voorstelling van karaktertekens met verschillende letters, leestekens, stuurcodes en cijfers. De cijfers gaan van 0 t/m 127. Wanneer je een 128e cijfer wil toevoegen, verandert de 'normale' 7-bits-code naar een 8-bits-code. Door een achtste bit toe te voegen, kun je meerdere tekens en letters gebruiken en ontstaat er een uitgebreide ASCII-code tot 256 tekens.
